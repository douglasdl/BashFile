# BashFile
Bash File Commands

## Navigation
Change directory
<br>
<code>
    cd <folder>
</code>
<br>

Change directory (return 1 level)
<br>
<code>
    cd ..
</code>
<br>


Present working directory
<br>
<code>
    pwd
</code>
<br>

Open the Present working directory in the Explorer
<br>
<code>
    open .
</code>
<br>

## Listing Files
List files in the current directory
<br>
<code>
    ls
</code>
<br>

List files including hidden files
<br>
<code>
    ls -all
</code>
<br>

## Check my IP Address
Display your PC Hostname
<br>
<code>
    hostname
</code>
<br>

Display your PC IP (Windows)
<br>
<code>
    ipconfig
</code>
<br>

Display your PC IP (Linux)
<br>
<code>
    ip
</code>
<br>

Display your PC IP (Linux)
<br>
<code>
    hostname -I
</code>
<br>

Display your PC IP (Linux)
<br>
<code>
    ip addr show
</code>
