# BashFile
Bash File Commands

## Navigation
Change directory
<br>
<code>
    cd <folder>
</code>
<br>

Change directory (return 1 level)
<br>
<code>
    cd ..
</code>
<br>


Present working directory
<br>
<code>
    pwd
</code>
<br>

## Listing Files
List files in the current directory
<br>
<code>
    ls
</code>
<br>

List files including hidden files
<br>
<code>
    ls -all
</code>
<br>
